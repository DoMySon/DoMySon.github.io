  • 假想最坏的情况，每次拿数据前都会上锁(可上锁的情况下,否则等待) 直到修改完数据。

  • 优点:

    1. 在竞争激烈条件下更有优势

    2. 当锁住的不止有一个变量的情况下

  • 缺点:

    1. 即乐观锁的优点
